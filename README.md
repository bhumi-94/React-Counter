# 🌿 React Counter App

✨ A minimal, elegant, and interactive counter built using React.
Designed with simplicity and smooth user experience in mind — perfect for beginners exploring state management.

---

## 📌 Preview

A clean UI with simple controls to:

* ➕ Increase count
* ➖ Decrease count

## 🛠️ Tech Stack

* ⚛️ **React.js** – Frontend library
* 🧠 **useState Hook** – State management
* 🎨 **CSS ** 
* 🌐 **Vercel** – Deployment

---

## ⚙️ How It Works

1. The counter value is stored in a state variable
2. Buttons trigger functions to update the state
3. React re-renders the UI instantly when state changes
